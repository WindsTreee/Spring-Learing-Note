request.getSession().setAttribute("USER_CONTEXT",user);

在Session中设置user变量，变量名为USER_CONTEXT,即以键值对的形式存储



request.getSession().removeAttribute("USER_CONTEXT");

移除Session中名为USER_CONTEXT的变量


dao文件中函数有多个变量需要传入.xml映射文件时，要使用@Param指定.xml中的变量名，不然即使变量名和.xml中的#{}里的名称一致也不能使用


int updatelastvisit(@Param("username") String username, @Param("lasttime") Date lasttime);

配置基于注解的事务时，要注意在xmlns和xsi中配置相应的tx命名空间
